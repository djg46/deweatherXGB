# deweatherXGB
deweatherXGB is an R package for meteorological normalisation of air quality data using XGBoost-based gradient boosted regression models. It can be installed using ```devtools::install_github("djg46/deweatherXGB")``` and is archived at [Zenodo DOI].

deweatherXGB was developed as part of doctoral research at the University of Leicester (Gregg, 2026), and is based on the [deweather package](https://github.com/davidcarslaw/deweather) and extends it with an XGBoost-based implementation.
